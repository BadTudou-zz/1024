#1024#

1024是一个基于**JAVA**的小游戏。

##游戏规则##
1. 可将活动方块向**上，下，左，右**4个方向移动
2. 相同数字的方块碰撞到一起可合并为一个方块，合并的方块数字为2者之和

##操作##
**单击**:选中目标方块
**移动**:*上*(W),*下*(S),*左*(A),*右*(D)
**重新开始**:空格(Space)
